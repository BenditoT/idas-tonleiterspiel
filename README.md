# Idas Tonleiterspiel

Eine kleine Web-App zum Üben von Dur- und Moll-Tonleitern auf der Klaviatur.

**Live:** https://benditot.github.io/idas-tonleiterspiel/

## Features

- 24 Tonarten (12 Dur + 12 Moll natürlich)
- Klaviersound, Streak-Zähler, Punktestand
- iPhone-optimiert, PWA-fähig

## Struktur

- `index.html` — Landing Page
- `spiel.html` — Die eigentliche Übungs-App

## Deployment

GitHub Pages via `.github/workflows/deploy.yml` — automatisch bei Push auf main.
